# Code of Conduct

This project follows the **Contributor Covenant, version 2.1**.

The full, authoritative text lives at the project's canonical URL:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

A plain-text mirror is also available at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt

By participating in this project — opening issues, submitting pull requests, commenting, reviewing — you agree to abide by the terms at the URL above.

## Reporting

To report behavior you believe violates the Code of Conduct, email the maintainers at **conduct@blavity.com**. Reports are handled privately and acknowledged within 5 business days.

For security vulnerabilities (not conduct concerns), use [GitHub Security Advisories](https://github.com/blavity/terraform-provider-kinsta/security/advisories/new) instead — see [SECURITY.md](SECURITY.md).

## Why a URL reference

We intentionally do not vendor the full Contributor Covenant text into this repo. The canonical URL is stable, versioned, and centrally maintained by the Contributor Covenant project. Linking to it keeps every project on the same, up-to-date text without per-repo drift.
